# AutomationExercise Playwright Tests

## Setup
1. Clone repo:
   git clone https://github.com/itznieru/automationexercise.git
   cd automationexercise

2. Install dependencies:
   npm install

3. Install Playwright browsers:
   npx playwright install


## Testing
1. allbrowser:
   runs test in all browsers

2. chrome:
   runs test in chrome
    
3. firefox:
   runs test in firefox
     
4. webkit:
   runs test in safari
